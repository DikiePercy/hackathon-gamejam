# KRIT

Сводка по критериям геймджема на основе текущего состояния репозитория (по коду и сценам).

## Что уже реализовано

### Геймдизайн / нарратив
- [x] Через механики читается идея защиты поезда в рейсе: бой на вагонах, посадка врагов, смерть/респавн, депо и переходы между сценами.
- [x] Есть играбельный вертикальный срез (меню -> старт рейса -> бой -> смерть/перезагрузка).

### Арт
- [x] В проекте подключен ключевой арт окружения/персонажей/объектов (вагоны, рельсы, персонажи, эффекты, параллакс).
- [x] Во многих местах используются рабочие заглушки/ассеты, позволяющие играть без полного финального арта.
- [x] Ключевые локации и игровые объекты уже визуально представлены (main, depot, train, wagons, menu).

### Техническое
- [x] Игра запускается как рабочий прототип со сценами меню, рейса и депо.
- [x] Можно отыграть игру без части контента (через текущие ассеты/заглушки).
- [x] Реализована система сохранений: new save, load selected, autosave, выбор слота.
- [x] Реализована загрузка последнего сейва после смерти игрока.
- [x] Реализованы базовые боевые/рейдовые механики: спавн волн врагов, стрельба, перезарядка, контактный урон, посадка врагов на поезд.

## Частично реализовано

### Геймдизайн / нарратив
- [ ] Все концовки и ключевые выборы: частично, полноценная система ветвления/концовок в явном виде не завершена.
- [ ] Объем 5-10 минут контента: частично, вертикальный срез есть, но насыщенность и длительность требуют отдельного плейтеста.

### Арт
- [ ] Полная консистентность арта и единый стиль: частично, ассеты из разных паков.
- [ ] Полностью адекватный нейминг всех файлов: частично, встречаются технические/смешанные названия.

### Техническое
- [ ] Базовый прототип со всеми концовками и ключевыми выборами: частично/в работе.

## Короткий итог
С технической стороны уже есть рабочий прототип с игровым циклом, боевой частью и сейвами. Основные недостающие блоки до полного соответствия таблице: нарративные концовки/выборы, формальная диалоговая система и командно-организационные артефакты.
