 ШАГ 1: Скопируй файлы
━━━━━━━━━━━━━━━━━━━━━━━━━━

Из этой папки скопируй каждый файл в соответствующее место проекта:

  classes_main-person.gd  →  res://classes/main-person.gd     (ЗАМЕНИТЬ)
  classes_bullet.gd       →  res://classes/bullet.gd           (НОВЫЙ)
  characters_Player.tscn  →  res://characters/Player.tscn      (ЗАМЕНИТЬ)
  characters_Bullet.tscn  →  res://characters/Bullet.tscn      (НОВЫЙ)
  train_Wagon.tscn        →  res://scenes/train/Wagon.tscn     (НОВЫЙ)
  train_train.gd          →  res://scenes/train/train.gd       (ЗАМЕНИТЬ)
  train_train.tscn        →  res://scenes/train/train.tscn     (ЗАМЕНИТЬ)


━━━━━━━━━━━━━━━━━━━━━━━━━━
 ШАГ 2: Добавь кнопки управления (Input Map)
━━━━━━━━━━━━━━━━━━━━━━━━━━

В Godot: верхнее меню → Project → Project Settings → вкладка Input Map

Нажми "Add New Action" и добавь ДВА новых действия:

  Действие: jump    → клавиша: Space
  Действие: shoot   → клавиша: Z  (или любая другая)

Как добавить клавишу: справа от названия действия нажми "+"
→ выбери "Key" → нажми нужную клавишу → нажми OK


━━━━━━━━━━━━━━━━━━━━━━━━━━
 ШАГ 3: Подключи сигналы в Player.tscn
━━━━━━━━━━━━━━━━━━━━━━━━━━

Открой res://characters/Player.tscn

В дереве слева найди узел LadderDetector (Area2D)
Справа вверху нажми вкладку "Node" (рядом с "Inspector")
В списке сигналов найди:
  area_entered  → двойной клик → в поле "Receiver Method" напиши:
				  _on_ladder_detector_area_entered → Connect
  area_exited   → двойной клик → напиши:
				  _on_ladder_detector_area_exited  → Connect


━━━━━━━━━━━━━━━━━━━━━━━━━━
 ШАГ 4: Назначь сцены в инспекторе
━━━━━━━━━━━━━━━━━━━━━━━━━━

А) Открой res://scenes/train/train.tscn
   Нажми на узел Train в дереве
   В Inspector справа найди поле "Wagon Scene"
   Перетащи туда файл res://scenes/train/Wagon.tscn

Б) Открой res://characters/Player.tscn
   Нажми на узел Player
   В Inspector найди поле "Bullet Scene"
   Перетащи туда файл res://characters/Bullet.tscn


━━━━━━━━━━━━━━━━━━━━━━━━━━
 ШАГ 5: Добавь игрока в главную сцену
━━━━━━━━━━━━━━━━━━━━━━━━━━

Открой res://scenes/main/main.tscn
Вверху редактора: кнопка "Instantiate Child Scene" (иконка цепочки)
→ выбери res://characters/Player.tscn → Open

Перемести игрока (Player) в сцене на крышу локомотива


━━━━━━━━━━━━━━━━━━━━━━━━━━
 Управление в игре
━━━━━━━━━━━━━━━━━━━━━━━━━━

  A / D  или  ← →   — ходить
  W  или  ↑          — прыгать / лезть на лестницу вверх
  S  или  ↓          — слезть по лестнице вниз
  Space              — прыжок
  Z                  — выстрел


━━━━━━━━━━━━━━━━━━━━━━━━━━
 Как работает персонаж
━━━━━━━━━━━━━━━━━━━━━━━━━━

  GROUND  ── нажать W у лестницы ──►  CLIMBING  ── доехал до верха ──►  ROOF
	▲                                      │                               │
	└────────── доехал до низа ────────────┘      нажать S у лестницы ──► CLIMBING
