# Godot Unit Tests

Этот каталог содержит простую тестовую инфраструктуру для запуска проверок прямо внутри Godot.

## Как запускать
1. Открой `tests/run_tests.tscn` в Godot.
2. Нажми Play.
3. Результаты будут выведены в Output/Debugger.

## Что тестируется
- `MainPerson`:
  - начальные патроны
  - выстрел уменьшает количество патронов
  - перезарядка запускается при пустой обойме
  - завершение перезарядки восстанавливает патроны
- `Enemy`:
  - начинает перезаряжаться, если пытается стрелять при пустом магазине
  - переключается на нож, если перезарядок больше нет
- `GameManager`:
  - флаг `has_shotgun`
  - изменение золота

## Примечания
- Тесты используют `res://tests/dummy_bullet.tscn`, чтобы не создавать настоящую пулю в игровом уровне.
- Если нужно больше проверок, добавь новые файлы `tests/test_*.gd` и зарегистрируй их в `tests/test_runner.gd`.
